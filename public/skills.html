CSS & ECMAScript

CSS (Cascasing Style Sheets) is a language used to describe how HTML elements are to be displayed to the user. It came about as a time saving method as many elements across different pages can all use the same unified styling, rather than styling each element individually.

ECMAScript is a specification created by Ecma International, and was created to form a standardised Javascript language. Over its lifetime, it has also been known as 'Mocha', 'LiveScript', and finally 'JavaScript'.
